Excercise to start GitHub Collaboration.

----------Assigment-----------------------------

1. The Chef -> Create a repository and invite members as collaborators
2. The Chef in the master branch will initialize an empty text file
3. The Chef will create a develop branch 
    The .txt file must have
    -> The name of the Restaurant
    -> What kind of menu it is
    -> A Drinks item with available Drinks

Every member of the group will be to add an item in the menu.
    -> for every member there will be a branch from starting from develop 
    -> The name of the branch will be "feature/name-course" and add the menu item

The complete work will have to be presented in develop, averyone will open a pull Request to develop.
The Chef's job is to merge the branches and fix the complete menu
